‚úèÔ∏è 2. Edit Property Listings

Open property-list.html in any code editor (we recommend VS Code or Sublime Text).

Find a code block like this:

To modify this:

  • Update title: Change the text inside <h4>
  • Update location: Change the text inside <span>
  • Update price: Change the amount inside <h5>
  • Change link: Update the href to the correct detail page (e.g., property-detail.html)
